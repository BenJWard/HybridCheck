Use HybRIDS in R scripts

HybRIDS is an open source package of R code, and so if you are familiar with R you can install the package in several ways and you can get programming ans scripting R code with it right away. For example you could install the latest version with devtools: install.packages("devtools") library(devtools) install_github("Ward9250/HybRIDS", build_vignettes=TRUE) library(HybRIDS)
